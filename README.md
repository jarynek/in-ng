# sao-in-ng

> my tiptop SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao in-ng my-project
in-ng my-project (new?)
```

### From git

```bash
sao jarek/sao-in-ng my-project
```
